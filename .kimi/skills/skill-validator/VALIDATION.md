# 验收清单

## 验证环境准备
- [ ] 准备待验证的 skill（如 atomic-commits）
- [ ] 确认该 skill 有 VALIDATION.md
- [ ] 准备验证环境（如 git 更改、截图文件等）

## 自然语言触发测试

| 触发词 | 预期行为 | 通过 |
|--------|----------|------|
| "验证技能" | 首先读取本 skill，然后指导验证流程 | ☐ |
| "测试技能" | 提供验证方法和步骤 | ☐ |
| "检查技能效果" | 指导如何验证 skill 是否生效 | ☐ |

## 功能验收清单

### 读取流程
- [ ] **首先读取 skill-validator/SKILL.md 本身**
- [ ] 然后读取被验证 skill/SKILL.md
- [ ] 然后读取被验证 skill/VALIDATION.md

### 验证指导
- [ ] 指导制定测试计划
- [ ] 指导准备验证环境
- [ ] 强调依次调用子 agent（禁止并行）
- [ ] 强调自然语言调用（不提 skill）

### 验收观察
- [ ] 指导观察子 agent 是否自动读取 skill
- [ ] 指导对比行为和规范
- [ ] 指导按 VALIDATION.md 逐项检查

### 总结评分
- [ ] 指导给出评分
- [ ] 指导输出验证报告

## 评分标准

- 全中（100%）：所有检查项通过
- 良好（80%）：核心功能正常，部分指导不够详细
- 及格（60%）：基本能指导验证，缺少关键步骤
- 失败（<60%）：没有首先读取本 skill，或指导有误

## 硬性要求

**必须首先读取 skill-validator 本身，否则视为失败！**
