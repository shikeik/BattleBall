# 验收清单

## 验证环境准备
- [ ] 确认项目根目录存在或可以创建 `TODO.md` 文件
- [ ] 准备测试用的任务数据

## 自然语言触发测试

| 触发词 | 预期行为 | 通过 |
|--------|----------|------|
| "管理待办" | 询问具体要执行的操作 | ☐ |
| "创建任务" | 询问任务详情并创建 | ☐ |
| "查看todo" | 列出所有任务 | ☐ |
| "添加任务" | 询问任务详情并添加 | ☐ |
| "完成任务" | 询问任务编号并标记完成 | ☐ |
| "跟踪进度" | 显示任务统计报告 | ☐ |

## 功能验收清单

### 文件操作
- [ ] 能自动检测项目根目录的 `TODO.md`
- [ ] 文件不存在时能自动创建
- [ ] 能正确读取和解析现有任务

### 任务管理
- [ ] 能创建带标题、优先级的新任务
- [ ] 能列出所有任务并显示状态
- [ ] 能更新任务状态（待办/进行中/已完成）
- [ ] 能删除不需要的任务
- [ ] 支持按优先级筛选任务

### 输出格式
- [ ] 任务列表带编号显示
- [ ] 使用图标表示优先级（🔴高/🟡中/🟢低）
- [ ] 使用复选框表示完成状态（☐/☑）
- [ ] 统计报告显示完成率百分比

### 边界情况
- [ ] 空列表时友好提示
- [ ] 任务标题为空时拒绝创建
- [ ] 无效的任务编号时给出错误提示

## 评分标准

- 全中（100%）：所有检查项通过
- 良好（80%）：核心功能正常，边界情况部分通过
- 及格（60%）：基本功能可用，有明显问题
- 失败（<60%）：核心功能无法正常使用
