# 验收清单

## 验证环境准备
- [ ] 确认手机有截图文件（DCIM/Screenshots/*.jpg）
- [ ] 或有录屏文件（DCIM/ScreenRecorder/*.mp4）
- [ ] 或有日志文件（Download/game-log*.log）

## 自然语言触发测试

| 触发词 | 预期行为 | 通过 |
|--------|----------|------|
| "看截图" | 查找并读取最新截图 | ☐ |
| "看最新视频" | 查找并读取最新录屏 | ☐ |
| "看录屏" | 查找并读取录屏文件 | ☐ |
| "看日志" | 查找并读取日志文件 | ☐ |
| "看log" | 查找并读取日志 | ☐ |

## 功能验收清单

### 截图查看
- [ ] 使用 Shell 查找最新截图（ls -t）
- [ ] 使用 ReadMediaFile 读取截图
- [ ] 分析截图内容

### 视频查看
- [ ] 使用 Shell 查找最新视频
- [ ] 使用 ReadMediaFile 读取视频
- [ ] 分析视频内容

### 日志查看
- [ ] 使用 Shell 查找最新日志
- [ ] 使用 ReadFile 读取日志内容
- [ ] 分析日志内容

### 输出格式
- [ ] 显示文件路径
- [ ] 描述文件内容
- [ ] 提供相关分析

## 评分标准

- 全中（100%）：所有检查项通过
- 良好（80%）：核心功能正常，部分文件类型支持不好
- 及格（60%）：基本能查找和读取，分析不够详细
- 失败（<60%）：无法正确查找或读取文件
